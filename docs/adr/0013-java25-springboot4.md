# ADR 0013: Java 25 and Spring Boot 4

## Status
Accepted

## Decision

Use Java 25 and Spring Boot 4.

## Reason

- modern features
- long-term support

## Implementation Notes

- pin Java 25 in CI and local tooling
- keep Spring Boot dependency versions aligned across modules

## References

- docs/architecture.md

## Consequences

- requires compatible tooling
